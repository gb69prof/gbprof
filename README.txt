gbprof - Laboratorio didattico digitale

Sito statico pronto per essere caricato su GitHub Pages o su un normale hosting.

File principale: index.html
Cartelle: assets/css, assets/js, assets/img

Pagine incluse:
- Home
- Filosofia
- Letteratura
- Storia
- PWA
- Mappe
- 3D/VR
- Collaborazione
- Archivio GitHub
- Progetti
- Contatti

Note:
- Le immagini generate sono usate come copertine delle sezioni.
- Le pagine Archivio, Progetti e Contatti contengono segnaposto da completare con link reali.
- Il sito include manifest e service worker: può comportarsi come PWA se servito da HTTPS.
